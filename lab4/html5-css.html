HTML5

  • HTML stands for Hyper Text Markup Language. HTML is the standard markup language for creating Web pages.
  • HTML describes the structure of a Web page.
  • HTML consists of a series of elements. HTML elements tell the browser how to display the content.
Element Was it used? Where was it used?
header task_alt Indicates header (usually with logos). Used at the beginning of each page.
nav task_alt Indicates navigation menu with links to pages (there should only be one per page). Used at the menu.
main cancel Indicates main and unique content, which is not repeated.
article cancel Indicates a portion of the document that can be viewed independently.
section task_alt Specifies a portion of the content, used in conjunction with tags. Used at the body, specifying its content.
aside cancel Indicates annotations, advertising, or other disconnected subjects.
footer task_alt Indicates section footer (with copyright). Used at the end of each page.
details cancel Defines additional details that the user may hide, contained in a summary element.
meter cancel Defines a static scale measure within a range.
progress cancel Define a progress bar of a task.
abbr cancel Indicates the meaning of an abbreviation in conjunction with the title attribute.
address cancel Contact information for the author.

CSS

  • CSS stands for Cascading Style Sheets.
  • CSS describes how HTML elements are to be displayed on screen, paper, or in other media.
  • CSS saves a lot of work. It can control the layout of multiple web pages all at once.
